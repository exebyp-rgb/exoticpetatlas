## Design Specifications

### Color Palette
- **Background**: Soft pastels (Warm beige #F5E6D3, Pale mint #B8E6C9, Dusty lavender #E5D4ED)
- **Accent**: Deep teal to midnight blue gradient (#4FD1C5 ‚Üí #2C7A7B ‚Üí #1A365D)
- **Details**: Purple-grey tones (#9B8AA3, #C9B8D4)

### Style Elements
- **Modern flat design** with subtle gradients
- **Minimal line work** for clean, scalable appearance
- **Reptilian/exotic features**: Vertical slit pupil, chitin texture on eyelids
- **Friendly but mysterious**: Intriguing without being scary
- **Professional**: Suitable for educational authority website

### Key Features
1. **Vertical slit pupil** - Suggests reptilian/exotic nature
2. **Subtle chitin/scale texture** - Around eyelids for organic detail
3. **Radial iris gradient** - Deep teal center fading to midnight blue
4. **Highlight spark** - Small white reflection for depth
5. **No gore/horror elements** - Curiosity-first, family-friendly

- **Authority**: An all-seeing guide and trusted resource
